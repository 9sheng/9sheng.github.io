## Scene

Sprint Planning — Discussing Priorities

- A: I think we should prioritize the API refactor this sprint. It's blocking two other teams.
- B: **Makes sense**. If it's a blocker, we should tackle it first.
- A: We might need to push the dashboard feature to next sprint *though*.
- B: **That's fair**. We can't do everything at once. **I'm on board** with that trade-off.
- A: Great. I'll update the sprint board.
- B: **Sounds good**. Let me know if you need help with the estimation.
